Call Taxi Booking Application:
This is a console-based Java application for booking taxis between points A to F.
The system automatically allocates taxis based on availability and rules.

Features:
1.Book a taxi

2.Automatic taxi allocation

3.Fare calculation

4.Display taxi details and total earnings
